# 🚢 Hundir la Flota

## Descripción
Implementación por consola del clásico juego de estrategia Hundir la Flota para dos jugadores.
El juego permite elegir entre enfrentarse a otro jugador o a la máquina en un tablero de 10x10,
colocando manualmente los barcos y disparando por turnos hasta hundir toda la flota rival.

## Contexto
Este proyecto nació como práctica de programación en Java, con el objetivo de trabajar arrays multidimensionales,
lógica de juego por turnos y separación de responsabilidades mediante métodos estáticos.
Incluye dos archivos: `Main.java` con la versión final del juego y `Test.java` con una versión de desarrollo
usada para pruebas durante el proceso de construcción.

## Tecnologías usadas
- **Java** — Lenguaje principal
- **Git / GitHub** — Control de versiones
